## 注释

* 常用三种注释风格:
    ```php
    echo 'str'
    // 111111111111

    # echo

    /*
        多行注释
    */
    ```

* 基于上面衍生出来的各种花式注释大法:
    ```php
    //======================================================================
    // CATEGORY LARGE FONT
    //======================================================================

    //-----------------------------------------------------
    // Sub-Category Smaller Font
    //-----------------------------------------------------

    /*
    * This is a detailed explanation
    * of something that should require
    * several paragraphs of information.
    */
    ```
